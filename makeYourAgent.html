Make your Agent

A simple tutorial to help the making of your agent.

Requirements

  • The python file of the agent must be in the agents_dir
  • You have to import the agents module: 
    from . agents import *
[...]
  • The agent class name of your agent must be "agent_file_name" + "Class"
  • The agent class must inerith Agent
    class agent_file_nameClass(Agent):
    def __init__(self):
        Agent.__init__(self)
        [...]
  • At the end of your init function you have to assign the attribut program with the function that update the status of tour agent
    class agent_file_nameClass(Agent):
    def __init__(self):
        Agent.__init__(self)

        [...]

        def program((status, bump)):
            pass

        self.progra = program

Events and actions

Your agent will receive a status and a variable bump. The status can be "Dirty" or "Clean". Bump instead can assume values "Bump" and "None". These variables refer to the actual position of the agent, in case of bump the target position can't be reached and the agent will remain in the same cell.

The possible action that an agent can do are:

  • "GoNorth"
  • "GoSouth"
  • "GoWest"
  • "GoEast"
To do an action during the environment updates you have to return these actions (strings) in the program function.
